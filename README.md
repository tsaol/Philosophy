# Philosophy — 用简单的话讲哲学

> Meditations：在忙碌的生活中，停下来想一想。

这个仓库试图用最简单的语言，讲清楚那些影响了人类几千年的大问题。每篇文章都很短，像一次散步时的思考。

## 目录

### [西方哲学](western.md)

按时间顺序：苏格拉底 → 柏拉图 → 亚里士多德 → 斯多葛学派 → 笛卡尔 → 康德 → 尼采

| 时期 | 哲学家 | 一句话 |
|------|--------|--------|
| 前469年 | 苏格拉底 | 承认无知，才是智慧的开始 |
| 前427年 | 柏拉图 | 你看到的世界，可能只是影子 |
| 前384年 | 亚里士多德 | 幸福不是感觉，是一种活法 |
| 前300年 | 斯多葛学派 | 控制你能控制的，接受你不能控制的 |
| 1596年 | 笛卡尔 | 怀疑一切，直到找到不可怀疑的 |
| 1724年 | 康德 | 做事之前问自己：如果人人都这么做呢？ |
| 1844年 | 尼采 | 上帝死了，你得自己找到活着的意义 |

### [东方哲学](eastern.md)

按时间顺序：老子 → 孔子 → 孙子 → 庄子 → 佛学思想 → 禅宗

| 时期 | 哲学家 | 一句话 |
|------|--------|--------|
| 前571年 | 老子 | 水最柔软，却能穿透岩石 |
| 前551年 | 孔子 | 做人要有规矩，但规矩的核心是爱 |
| 前544年 | 孙子 | 不战而屈人之兵，善之善者也 |
| 前369年 | 庄子 | 鱼快不快乐，你怎么知道？ |
| 前5世纪 | 佛学思想 | 苦的根源是执着 |
| 公元5世纪 | 禅宗 | 别想了，去体验 |

### [哲学核心问题](topics.md)

| 主题 | 核心问题 |
|------|----------|
| 存在与本体论 | 为什么有东西存在，而不是一片虚无？ |
| 认识论 | 你怎么知道你知道的是对的？ |
| 伦理学 | 什么是对，什么是错？ |
| 美学 | 为什么有些东西让你觉得美？ |
| 逻辑学 | 怎样的推理才是靠谱的？ |
